Tentang Metode

Penjelasan singkat mengenai Gray Level Co-occurrence Matrix (GLCM) dan SVM

1. Gray Level Co-occurrence Matrix (GLCM)

GLCM adalah matriks statistik orde dua yang menggambarkan seberapa sering pasangan nilai keabuan piksel muncul bersama pada jarak d dan sudut θ tertentu dalam sebuah citra. GLCM digunakan untuk mengekstraksi informasi tekstur dari citra, yang sangat berguna untuk membedakan motif batik dengan pola tekstur berbeda.

2. Tahapan Pra-pemrosesan

  1. Citra diubah menjadi grayscale (skala keabuan).
  2. Citra di-resize menjadi {{ meta.img_size[0] }}x{{ meta.img_size[1] }} piksel.
  3. Nilai keabuan dikuantisasi menjadi {{ meta.levels }} level untuk mengurangi kompleksitas komputasi GLCM.

3. Parameter GLCM yang Digunakan

  • Jarak (distance): {{ meta.distances|join(', ') }} piksel
  • Sudut (angle): 0°, 45°, 90°, 135°
  • Level keabuan: {{ meta.levels }}

4. Fitur Tekstur (Properti GLCM)

Dari setiap matriks GLCM, dihitung enam properti statistik berikut:

Contrast = Σ (i-j)² * P(i,j) Dissimilarity = Σ |i-j| * P(i,j) Homogeneity = Σ P(i,j) / (1 + (i-j)²) Energy = √ASM ASM = Σ P(i,j)² Correlation = Σ (i-μi)(j-μj)P(i,j) / (σi σj)

Setiap properti dihitung untuk setiap kombinasi jarak dan sudut, menghasilkan total {{ meta.feature_names|length }} fitur per citra.

5. Klasifikasi dengan SVM

Vektor fitur GLCM dinormalisasi menggunakan StandardScaler, kemudian diklasifikasikan menggunakan Support Vector Machine (SVM) dengan kernel RBF. Model dilatih menggunakan {{ meta.n_train }} citra latih dan diuji dengan {{ meta.n_test }} citra uji, mencapai akurasi sebesar {{ "%.2f"|format(meta.test_accuracy * 100) }}% pada {{ meta.classes|length }} kelas motif batik.
